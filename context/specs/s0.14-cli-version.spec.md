# S0.14: CLI Version Flag

**Type:** Polish
**Depends on:** None

---

## What to Build

Add `--version` / `-V` to both `device` and `attester` CLIs. Clap reads the version from `Cargo.toml` automatically via `version = env!("CARGO_PKG_VERSION")`.

### `device/src/main.rs`

```rust
#[derive(Parser)]
#[command(
    name = "device",
    about = "HardTrust device CLI — generate key and emit readings",
    version,   // ← add this
)]
struct Cli { ... }
```

### `attester/src/main.rs`

```rust
#[derive(Parser)]
#[command(
    name = "attester",
    about = "HardTrust attester CLI — register and verify devices",
    version,   // ← add this
)]
struct Cli { ... }
```

That's it. Clap's `version` attribute macro picks up `CARGO_PKG_VERSION` from the crate's `Cargo.toml` automatically.

**Expected output:**
```
$ device --version
device 0.1.0

$ attester --version
attester 0.1.0
```

## Validation Criteria

- [ ] `device --version` prints `device 0.1.0`
- [ ] `attester --version` prints `attester 0.1.0`
- [ ] `just ci` passes
