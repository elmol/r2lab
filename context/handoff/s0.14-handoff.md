# Handoff: S0.14 — CLI Version Flag

## Context

Both `device` and `attester` binaries had no `--version` flag. Users couldn't verify what version was installed.

## Spec

Read `docs/specs/s0.14-cli-version.spec.md` for full details.

## Task

Add `version` to the `#[command]` attribute in both CLIs. Clap reads `CARGO_PKG_VERSION` automatically.

### `device/src/main.rs`

```rust
#[derive(Parser)]
#[command(
    name = "device",
    about = "HardTrust device CLI — generate key and emit readings",
    version,   // ← add this
)]
struct Cli { ... }
```

### `attester/src/main.rs`

```rust
#[derive(Parser)]
#[command(
    name = "attester",
    about = "HardTrust attester CLI — register and verify devices",
    version,   // ← add this
)]
struct Cli { ... }
```

## Branch

`feat/s0.14-cli-version`

## Acceptance

- `device --version` prints `device 0.1.0`
- `attester --version` prints `attester 0.1.0`
- `just ci` passes
