# Handoff: S0.10 — Install Script

## Context

A `install.sh` at the repo root that detects the platform, downloads the correct binary from GitHub Releases, verifies SHA256, and installs it.

## Spec

Read `docs/specs/s0.10-install-script.spec.md` — contains the complete script content.

## Task

1. Create `install.sh` at the repository root with the content from the spec
2. **Replace `OWNER` in the script** with the actual GitHub username/org
3. Make it executable in git: `git update-index --chmod=+x install.sh`
4. Validate: `bash install.sh` on Ubuntu installs `attester`, on Raspbian installs `device`

## Branch
`feat/s0.10-install-script`

## Acceptance
- File is executable in git (`100755` mode)
- Platform detection works for: Linux x86_64, Linux armv7l, macOS arm64
- SHA256 verification catches corrupted downloads
- `INSTALL_DIR`, `HARDTRUST_VERSION`, `HARDTRUST_BINARY` env vars respected
