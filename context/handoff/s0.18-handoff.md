# Handoff: S0.18 — Fix Install Scripts Trap Unbound Variable

## Context

Both install scripts fail on exit with `bash: línea 1: tmp: variable sin asignar`. The `tmp` variable is `local` inside `main()` but the trap fires after `main()` returns, when `tmp` is out of scope. With `set -u`, bash errors on the unbound reference.

## Spec

Read `docs/specs/s0.18-install-trap-fix.spec.md` for full details.

## Task

### Branch

```bash
git checkout -b fix/s0.18-install-trap-fix
```

### Step 1 — Fix install-device.sh

Add module-scope cleanup before `main()`:

```bash
TMP_DIR=""
cleanup() { [ -n "${TMP_DIR}" ] && rm -rf "${TMP_DIR}"; }
trap cleanup EXIT
```

Inside `main()`, replace:

```bash
local tmp; tmp="$(mktemp -d)"; trap 'rm -rf "${tmp}"' EXIT
```

with:

```bash
TMP_DIR="$(mktemp -d)"
```

Replace all `${tmp}` references in `main()` with `${TMP_DIR}`.

### Step 2 — Fix install-attester.sh

Apply the exact same fix.
